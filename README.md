# 羊毛自动化 

自动化的羊毛目前一般有两种方案。一种是Android的Autojs，这种APP的原理是基于模拟用户点击，在手机端操作。 另一种是青龙面板，一般部署在服务器上，这种主要的原理是模拟请求。

## Autojs

本项目旨在总结网友提供的Autojs脚本，以便能更好的在Android上完成一些自动化的操作，非常感谢广大的github友友们。
